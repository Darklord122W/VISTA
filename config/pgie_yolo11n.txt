# IMPORTANT: this is a GLib GKeyFile. It only supports comments on their OWN
# line (a line starting with '#'). A trailing "key=value  # note" would make the

# So every comment here is on its own line.
#
# Paths are relative to THIS file's directory (nvinfer resolves model paths
# relative to the config file), so the app runs correctly from any directory.
#
# batch-size below is a fallback; pipeline_builder.py overrides the nvinfer
# batch-size property at runtime to equal the active camera count (the ONNX was
# exported with --dynamic, so one engine serves batch 1..N). The engine is
# auto-named per batch/precision, e.g. yolo11n.onnx_b4_gpu0_fp16.engine.
# =============================================================================
